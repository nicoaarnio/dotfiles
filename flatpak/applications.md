## ![Discord](/flatpak/icons/16x16/discord.png) Discord
**Messaging, Voice, and Video Client**

Install:

```
flatpak install flathub com.discordapp.Discord
```

Run:

```
flatpak run com.discordapp.Discord
```
<br>
